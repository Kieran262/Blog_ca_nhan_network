> Khi học JavaScript, mình đã tò mò về cách trình duyệt giao tiếp với máy chủ. Bài viết này là những ghi chép của mình.

## HTTP là gì?

HTTP (HyperText Transfer Protocol) là giao thức để trình duyệt và máy chủ giao tiếp. Mình hiểu nó giống như ngôn ngữ chung giữa client và server.

## JavaScript trong trình duyệt

JavaScript chạy trong trình duyệt có thể gửi request HTTP đến server. Mình học về `XMLHttpRequest` và sau đó là `fetch()` API.

## Request và Response

Mỗi request có:
- Method: GET, POST, PUT, DELETE...
- URL: Địa chỉ của tài nguyên
- Headers: Thông tin bổ sung
- Body: Dữ liệu gửi kèm (nếu có)

## Kết luận

Hiểu về HTTP giúp mình viết JavaScript tốt hơn khi làm việc với API và server.
